Find & Merge Duplicate Data.

Key Features:

  • Allows you to merge the duplicate data.
  • Find Duplicate data from any model. User have option for find duplicate data based on the selected field.
  • User can merge the duplicate data of any model like Product, Partners, Product Category as well as any Custom model.
  • It will update the duplicate record reference with original record at all places in your odoo system.
  • It gives the option for what action you want to perform on the duplicate record like Delete, Archived or Nothing.
  • Specific user can merge the duplicate data.
See Video

Overview

  • From Users menu, you can give access for who can merge the duplicate data.
  • From top Action, user can see the option for Find Duplicate Data. User have this option in all other model too.
  • In opened pop-up user have to select the field. Based on selected field(s), it check the duplicate data.
  • Result of the Duplicate data based on the selected field(s).
  • Select record that you want to merge and from top Action menu select the action Merge Duplicate Data. If user have an access rights of Merge, then only user can see this Action.
  • In opened window, User have to select the Action for the duplicate record. Also select the checkbox for the Original Record.
  • After merge, based on selected option in wizard for the duplicate record, the record will remove or archived or remain into the database.
  • User also have another option for merge data. For this option go to menu Settings > Custom Shortcuts > Merge Duplicate Data. In opened pop-up window, you have to select Model, then Original record and Duplicate record. Also select action option for duplicate record. After click on Merge Data button, It will replace the reference of the Duplicate record with Original record at all the places in odoo system.
