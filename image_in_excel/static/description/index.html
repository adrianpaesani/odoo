Export Images instead of Base64

Advantage:-It will help in exporting images instead of Base64 when you Export data in Excel

By Default, How are product images exported to Excel file from Odoo?

While Exporting product information in Odoo to excel file, the product images are automatically encoded to Base64 format.

The file, thus produced contains the Base64 code in the form of letters and numbers instead of the image preview.

So, What Does Show Images In Excel Module Do?

The module overrides the inherent property of Odoo to export Base64 code of images while exporting them from Odoo to excel file.

It is designed to preserve and present the image in the original format so the image can be viewed in the file for better visual presentation to the viewer.

Features:

  • The module exports the original Odoo product image to the excel file while exporting product data.
  • The module prevents the product images from being converted Base64 format while exporting it.
  • Once exported, the image column in the file shows the image preview of the product image instead of Base64 code.
